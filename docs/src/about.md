# About

codon is maintained by [misut](https://github.com/misut) and lives at
[github.com/misut/codon](https://github.com/misut/codon).

It is part of a small C++23 workspace:

- [exon](https://github.com/misut/exon) — package manager
- [intron](https://github.com/misut/intron) — toolchain manager
- [phenotype](https://github.com/misut/phenotype) — UI framework and
  WebGPU 2D renderer the reader is built on
- [tomlcpp](https://github.com/misut/tomlcpp) — TOML parser used to
  load `book.toml`
- [jsoncpp](https://github.com/misut/jsoncpp) — JSON parser / emitter
  used to round-trip `book.json`
